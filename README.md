# BelfieldSOPs
SOPs were originally .docx files. converted to JSON so that they can be version controlled in github. Then new code was added allowing the SOPs to be edited in a web page to ensure those not familiar with json structure don;t get confused. 
Installed Streamlit to PATH (in windows env)
to access the web interface need to run in terminal, point filename to the repo and usethe following command
streamlit run app.py
if streamlit is not added to PATH then you need to force the act use:  python -m streamlit run app.py
